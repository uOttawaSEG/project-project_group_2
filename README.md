## Overview
OTAMS (Online Tutoring Appointment Management System) is an Android application designed to 
connect university students with tutors for academic support.
The system allows students search for available tutors by course code, request specific time slots, 
and track their requests. Tutors publish their availability, specify which courses they can teach, 
and manage incoming student requests through an approval
